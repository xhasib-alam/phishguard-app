# PhishGuard

PhishGuard is an AI-powered phishing detection Android application designed to identify suspicious and malicious URLs.

## Features

* Phishing URL Detection
* Risk Score Analysis
* Confidence Score
* Detailed Security Findings
* Android Mobile Support
* Fast URL Scanning
